## Proje Amacı

Akilli_Mutfak_Asistani_ve_Spor_Asistani; kullanıcıların yapay zeka destekli tarif önerileri alabilmesini, günlük besin takibi yapabilmesini ve kişiselleştirilmiş spor planları oluşturabilmesini sağlayan bir mobil uygulamadır.Proje, React Native ve Expo altyapısı kullanılarak geliştirilmiş olup kullanıcıların sağlıklı yaşam alışkanlıklarını tek bir uygulama üzerinden yönetebilmesini amaçlamaktadır.

## Kurulum Adımları

### 1. Projeyi İndirme

```bash
git clone https://github.com/emirhandurmus-netizen/Akilli_Mutfak_Asistani_ve_Spor_Asistani
```

### 2. Proje Dizinine Girme

```bash
cd Akilli_Mutfak_Asistani_ve_Spor_Asistani
```

### 3. Gerekli Paketleri Kurma

```bash
npm install
```

### 4. Ortam Değişkenlerini Oluşturma

`.env.local` dosyası oluşturulur ve gerekli API anahtarları eklenir.

### 5. Uygulamayı Çalıştırma

```bash
npm run start:all
```
